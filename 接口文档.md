# 聊天应用后端接口文档

## 🔍 接口概览

### 用户端接口
- **用户管理**
    - POST `/user/login` - 用户登录
    - POST `/user/register` - 用户注册
    - GET `/user/user/UserInfo` - 获取用户信息
    - POST `/user/update` - 更新用户信息
    - POST `/user/updateAvatar` - 更新用户头像
    - GET `/user/avatar` - 查看用户头像

- **角色管理**
    - POST `/character/add` - 新增角色
    - POST `/character/update` - 修改角色
    - GET `/character/delete` - 删除角色
    - GET `/character/search` - 检索角色
    - GET `/character/isPublicOrNot` - 查询角色是否公开
    - POST `/character/isPublic` - 设置角色公开/私密
    - GET `/character/my-list` - 获取我的角色列表
    - GET `/character/square` - 角色广场
    - GET `/character/{id}` - 获取角色详情
    - POST `/character/follow` - 关注角色
    - GET `/character/my-and-follow` - 获取我的关注和创建的角色

- **AI 聊天**
    - POST `/ai/chat` - AI 对话核心接口
    - GET `/ai/chat/history/session/{sessionId}` - 获取指定会话的历史记录
    - GET `/ai/chat/sessions/{userId}/{characterId}` - 获取会话列表
    - POST `/ai/chat/new/{characterId}` - 创建新会话

- **群组聊天**
    - POST `/api/group/create` - 创建群组
    - POST `/api/group/join` - 加入群组
    - POST `/api/group/leave` - 退出群组
    - GET `/api/group/{groupId}/members` - 查询群成员
    - GET `/api/group/{groupId}/detail` - 查询群详情
    - GET `/api/group/{groupId}` - 按群号查询群信息
    - GET `/api/group/my` - 查询当前用户加入的群组
    - GET `/api/group/{groupId}/messages` - 查询群历史消息
    - POST `/api/group/messages/query` - 查询群历史消息(高级)

- **关注管理**
    - POST `/user/follow/follow` - 关注/取消关注角色
    - GET `/user/follow/status` - 查看关注状态
    - GET `/user/follow/list` - 查看关注列表
    - GET `/user/follow/count` - 查看角色被关注数量
    - GET `/user/follow/rank` - 关注排行榜

- **统计指标**
    - GET `/metrics/tokens/{memoryId}` - 查询指定会话的 Token 统计
    - GET `/metrics/tokens/daily` - 查询用户每日 Token 使用量

- **通用功能**
    - POST `/user/common/upload` - 通用文件上传
    - POST `/user/common/uploadCharacterAvatar` - 角色头像上传
    - POST `/user/common/uploadUserAvatar` - 用户头像上传

### 管理端接口
- **管理员管理**
    - POST `/admin/login` - 管理员登录
    - POST `/admin/register` - 管理员注册

- **仪表盘**
    - GET `/admin/dashboard/stats` - 获取仪表盘统计数据

- **用户管理**
    - GET `/admin/user/page` - 用户分页查询
    - POST `/admin/user/status/{status}` - 封禁/解封用户

## 📝 详细接口说明

### 1. 用户管理模块 (User)

#### 1.1 用户登录
- **接口**: `POST /user/login`
- **描述**: 用户登录认证，成功后返回JWT令牌。
- **参数**:
  ```json
  {
    "username": "testuser",
    "password": "123456"
  }
  ```
- **响应**:
  ```json
  {
    "code": 1,
    "msg": "success",
    "data": {
      "id": 1,
      "username": "testuser",
      "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
    }
  }
  ```

#### 1.2 用户注册
- **接口**: `POST /user/register`
- **描述**: 注册新用户。
- **参数**:
  ```json
  {
    "username": "newuser",
    "password": "123456"
  }
  ```

#### 1.3 获取用户信息
- **接口**: `GET /user/user/UserInfo`
- **描述**: 获取当前登录用户的详细信息。

#### 1.4 更新用户信息
- **接口**: `POST /user/update`
- **参数**:
  ```json
  {
    "id": 1,
    "username": "updatedName",
    "avatar": "http://..."
  }
  ```

#### 1.5 更新用户头像
- **接口**: `POST /user/updateAvatar`
- **参数**: `userId` (Long), `imageUrl` (String)

#### 1.6 查看用户头像
- **接口**: `GET /user/avatar`

### 2. 角色管理模块 (Character)

#### 2.1 新增角色
- **接口**: `POST /character/add`
- **参数**: `Character` 对象 JSON

#### 2.2 修改角色
- **接口**: `POST /character/update`
- **参数**: `Character` 对象 JSON (需包含 id)

#### 2.3 删除角色
- **接口**: `GET /character/delete`
- **参数**: `characterId` (Long)

#### 2.4 检索角色
- **接口**: `GET /character/search`
- **参数**: `name` (String, 可选), `personality` (String, 可选)

#### 2.5 查询角色是否公开
- **接口**: `GET /character/isPublicOrNot`
- **参数**: `characterId` (Long)

#### 2.6 设置角色公开/私密
- **接口**: `POST /character/isPublic`
- **参数**: `characterId` (Long)
- **说明**: 切换公开状态。

#### 2.7 获取我的角色列表
- **接口**: `GET /character/my-list`
- **响应**: 包含用户创建的角色列表。

#### 2.8 角色广场
- **接口**: `GET /character/square`
- **响应**: 获取所有公开的角色列表。

#### 2.9 获取角色详情
- **接口**: `GET /character/{id}`

#### 2.10 关注角色
- **接口**: `POST /character/follow`
- **参数**: `id` (Long)

#### 2.11 获取我的关注和创建的角色
- **接口**: `GET /character/my-and-follow`

### 3. AI 聊天模块 (Chat)

#### 3.1 AI 对话核心接口
- **接口**: `POST /ai/chat`
- **Content-Type**: `multipart/form-data` (支持图片上传)
- **参数**:
    - `memoryId` (String, 可选): 会话ID，为空则创建新会话
    - `userId` (String): 用户ID
    - `characterId` (String): 角色ID
    - `message` (String): 用户发送的消息
    - `imageFile` (File, 可选): 上传的图片文件
    - `character` (Object): 角色信息对象
- **响应**: AI 回复的文本内容 (Stream)

#### 3.2 获取指定会话的历史记录
- **接口**: `GET /ai/chat/history/session/{sessionId}`
- **参数**: `page` (int, 默认1), `size` (int, 默认100)

#### 3.3 获取会话列表
- **接口**: `GET /ai/chat/sessions/{userId}/{characterId}`
- **说明**: 获取用户与某角色的所有历史会话列表（左侧会话栏）。

#### 3.4 创建新会话
- **接口**: `POST /ai/chat/new/{characterId}`
- **响应**: 返回新的 `sessionId`。

### 4. 群组模块 (Group)

#### 4.1 创建群组
- **接口**: `POST /api/group/create`
- **参数**:
  ```json
  {
    "creatorId": 1,
    "name": "群组名称",
    "characterId": "AI角色ID",
    "description": "群描述"
  }
  ```

#### 4.2 加入群组
- **接口**: `POST /api/group/join`
- **参数**:
  ```json
  {
    "groupId": 1,
    "userId": 1,
    "nickname": "群昵称"
  }
  ```

#### 4.3 查询群成员
- **接口**: `GET /api/group/{groupId}/members`

#### 4.4 查询群详情
- **接口**: `GET /api/group/{groupId}/detail`

#### 4.5 查询历史消息
- **接口**: `GET /api/group/{groupId}/messages`
- **参数**: `page`, `size`

### 5. 关注模块 (Follow)

#### 5.1 关注/取消关注
- **接口**: `POST /user/follow/follow`
- **参数**: `id` (Long, 角色ID)

#### 5.2 查看关注状态
- **接口**: `GET /user/follow/status`
- **参数**: `id` (Long)

#### 5.3 查看关注列表
- **接口**: `GET /user/follow/list`

#### 5.4 关注排行榜
- **接口**: `GET /user/follow/rank`
